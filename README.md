# 🏥 PhysioCare Clinic Management System

A comprehensive, modern clinic management platform designed specifically for physiotherapy clinics. Built with **Laravel**, this system streamlines the workflow between administrators, physiotherapists, and patients.

## ✨ Features

### 👨‍💼 Admin Portal
- **Dashboard & Analytics:** Overview of daily operations, upcoming appointments, and patient statistics.
- **Patient Management:** Complete patient profiles including medical history, documents, and attendance tracking.
- **Doctor Management:** Manage staff profiles, specialized fields, and dynamic doctor availability schedules.
- **Treatment Calendar:** A centralized view of all active therapy plans and daily sessions across the clinic.
- **Financials:** Generate, manage, and track patient invoices.

### 👨‍⚕️ Doctor Portal
- **My Schedule:** Doctors can set and update their available working hours dynamically.
- **Clinical Dashboard:** View assigned patients, upcoming daily appointments, and clinical history.
- **Prescriptions & Notes:** Create digital doctor notes, prescribe exercises, and print professional PDFs directly from the browser.
- **Treatment Execution:** Update patient therapy session statuses (Completed, Cancelled, Pending).

### 👥 Public Patient Portal
- **Online Booking:** A beautiful, glassmorphic public booking interface.
- **Smart Availability:** The booking form automatically synchronizes with doctor schedules, ensuring patients can only book when a doctor is actively working.
- **Specialty Filtering:** Patients can browse doctors based on their clinical specialty.

## 🛠️ Technology Stack
- **Backend:** Laravel (PHP)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3 (Custom Design System), Vanilla JS, Blade Templates
- **Styling:** Premium modern UI with glassmorphism and tailored color palettes.

## 🚀 Getting Started

### Prerequisites
- PHP >= 8.1
- Composer
- MySQL Database (e.g., via XAMPP)
- Node.js & NPM (optional, for asset compiling)
